Założenia projektu:
1. Stworzenie reprezentacji logicznej kostki Rubika NxN,
2. Implementacja prezentacji graficznej kostki,
3. Stworzenie interfejsu użytkownika pozwalającego szybko i intuicyjnie operować kostką,
4. Implementacja algorytmu rozwiązującego kostki rubika NxN,
5. Implementacja układanek logicznych nie będących kostkami.

Spełnienie założeń:
1. Całkowicie spełnione,
2. Całkowicie spełnione,
3. Interfejs jest szybki dla kostki 3x3, dla większych trzeba używać myszki,
4. Algorytm działa jedynie w przypadku N = 1,
5. Założenie nie spełnione.

Wykorzystne technologie:
Java 1.8
JavaMonkeyEngine3 3.2.1-stable
Gradle 4.6

Napotkane problemy:
-Spójne połączenie reprezentacji logiczej i graficznej
-Opis teoretyczny obrotów i orientacji ciał w przestrzeni trójwymiarowej
